Scheduling System Setup Guide

This guide will help you set up an automated scheduling system for your tutoring website. Choose the option that best fits your needs.

Option 1: Cal.com (Recommended - Free)

Step 1: Create a Cal.com Account

  1. Visit cal.com
  2. Click "Get Started for free"
  3. Sign up with your email or Google account
  4. Choose a username (e.g., "alain-carpenter")

Step 2: Configure Your Availability

  1. Go to Settings → Availability
  2. Set your working hours
  3. Add buffer time between sessions
  4. Set minimum booking notice (e.g., 24 hours)

Step 3: Create Event Types

  1. Click "Event Types" → "New Event Type"
  2. Create different session types:
    • 60-minute Tutoring Session
    • 90-minute Extended Session
    • 30-minute Consultation
  3. For each event type, add:
    • Description
    • Location (Zoom, In-person, etc.)
    • Custom questions (Subject area, specific topics)

Step 4: Integrate with Your Website

Replace the placeholder in script.js line 166:

// Change this line:
const calUsername = 'YOUR_CAL_LINK';

// To your actual Cal.com username:
const calUsername = 'alain-carpenter'; // Your actual username

The scheduling widget will automatically load on your website!

Step 5: Email Notifications

  1. In Cal.com, go to Settings → Workflows
  2. Create automated emails for:
    • Booking confirmation
    • Reminder 24 hours before
    • Follow-up after session

Option 2: Calendly (Alternative)

Similar to Cal.com but with different features. Free plan includes:

  • One event type
  • Unlimited bookings
  • Calendar integration

To use Calendly instead, replace the Cal.com code in script.js with:

// Calendly integration
const calendlyUsername = 'your-calendly-username';
const script = document.createElement('script');
script.src = 'https://assets.calendly.com/assets/external/widget.js';
script.async = true;
document.body.appendChild(script);

script.onload = function() {
    Calendly.initInlineWidget({
        url: `https://calendly.com/${calendlyUsername}/60min`,
        parentElement: document.getElementById('cal-embed'),
        prefill: {},
        utm: {}
    });
};

Option 4: Google Calendar Appointment Scheduling

If you have a Google Workspace account:

  1. Enable appointment scheduling in Google Calendar
  2. Create booking pages for different session types
  3. Embed the booking page in an iframe

Add to your website:

<iframe src="YOUR_GOOGLE_BOOKING_PAGE_URL" 
        width="100%" 
        height="600" 
        frameborder="0"></iframe>
Important Setup Notes:
  • Connect your calendar (Google, Outlook, etc.) to prevent double-booking
  • Set up payment processing if needed (Stripe integration available)
  • Configure time zones correctly
  • Test the booking flow thoroughly before going live
